## 1. 채널

- 고 채널은 고루틴 간에 데이터를 주고 받을 수 있는 통로를 제공
- 채널 버퍼는 채널에 데이터를 전송할 때 버퍼에 데이터를 저장하고, 수신할 때 버퍼에서 데이터를 꺼내는 방식

``` go
func main() {
    // 채널 생성
    c := make(chan int)

    // 고루틴 생성
    go func() {
        // 채널에 데이터 전송
        c <- 1
    }()

    // 채널에서 데이터 수신
    fmt.Println(<-c)
}
```

## 3. 채널 닫기

- 채널을 닫으면 채널에 데이터를 전송할 수 없음

``` go
func main() {
    // 채널 생성
    c := make(chan int)

    // 고루틴 생성
    go func() {
        // 채널에 데이터 전송
        c <- 1
        // 채널 닫기
        close(c)
    }()

    // 채널에서 데이터 수신
    fmt.Println(<-c)
}
```
